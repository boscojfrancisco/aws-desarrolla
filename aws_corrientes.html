AWS Desarrolla

El Gobierno de la Provincia de Corrientes y Amazon Web Services unen fuerzas para impulsar el programa AWS Desarrolla, una iniciativa destinada a fortalecer el ecosistema tecnológico provincial mediante el acceso a infraestructura digital en la nube, capacitación y acompañamiento técnico. El programa permite que emprendedores, startups de base tecnológica de CORRIENTES puedan desarrollar, probar y escalar soluciones digitales utilizando tecnología cloud de nivel internacional, sin necesidad de inversión inicial en equipamiento o servidores.

AWS DESARROLLA

¿Qué es AWS Desarrolla?

El programa permite que emprendedores, startups, pymes, instituciones y organismos públicos de Corrientes puedan desarrollar, probar y escalar soluciones utilizando tecnología cloud de nivel internacional, sin necesidad de inversión inicial en equipamiento o servidores.

¿Cómo Participar?

Las personas u organizaciones interesadas podrán postular sus proyectos tecnológicos para acceder a los beneficios del programa AWS Desarrolla. Se priorizan iniciativas que desarrollen productos tecnológicos, utilicen o prevean utilizar AWS y presenten potencial de crecimiento o impacto.

A través de créditos promocionales y servicios AWS, los proyectos pueden:

  • Desarrollar aplicaciones web y móviles
  • Crear plataformas y servicios digitales
  • Implementar bases de datos y almacenamiento en la nube
  • Incorporar analítica de datos e inteligencia artificial
  • Escalar soluciones tecnológicas según demanda
  • Desarrollar prototipos o productos mínimos viables (MVP)
  • Facilitar el acceso a infraestructura tecnológica de primer nivel.
  • Fomentar la innovación y el desarrollo de nuevos productos y servicios.
  • Impulsar el crecimiento y la competitividad del ecosistema tecnológico local.
  • Brindar capacitación y acompañamiento para optimizar el uso de la nube.

Los créditos promocionales de AWS se aplican a la factura mensual de AWS del proyecto para cubrir el costo de los servicios elegibles. Los beneficiarios recibirán asistencia técnica para la activación y optimización de los créditos asignados.

Emprendedores tecnológicos, startups en etapa temprana o de crecimiento, pequeñas y medianas empresas (PyMEs) del sector TIC, instituciones educativas y centros de investigación, y organismos del sector público de la Provincia de Corrientes.

Contar con un proyecto tecnológico definido, tener o crear una cuenta activa de Amazon Web Services, y completar el formulario de postulación detallando el caso de uso y la arquitectura propuesta.
